# A Library the contains functions to count a word frequency in a string
# And to test whether a given string is palindrome or not

## Usage

- Install the package using the command
```
  go get abrishk26/a2sv-project-track/task2
  
```
- Import the package at the start of you code
```
  import (
    "abrishk26/a2sv-project-track/task2"
  )
```
- Use the functions
```
  count := task2.CountWordFreq("foo bar FOO BAR")

  if task2.IsPalindrome("madam") {}
```
